# 工作流
## 0x01 设计哲学
核心在于“流”；所有一切，都是为了信息尽可能少的信赖和限制下流动。
`信息->收集->整理->输出`
信息：
- 微博
- 知乎
收集：
微信---> send kindle
整理：
> 整理为知识库
- 不使用应用，不把数据绑定到某个应用中
- 使用文件夹，实现跨平台
输出：
学习的升华过程，增加对知识的理解，锻炼文件组织及表达勇
