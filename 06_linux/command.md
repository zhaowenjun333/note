# 基本命令

## cp
### cp原理：
cp 与 inode
1. 分配一个空闲的inode号， 在inode表中生成新条目，在目录中创建一个目录项，将名称与inode号关联
2. 拷贝数据生成新的文件







## rm
### rm 原理
rm 与 inode
1. 链接数递减，从而释放的 inode 号可以被重新使用，把数据块放在空闲列表中
删除目录项
2. 数据实际上不会马上被删除



## find命令

关于权限的解理
```
-perm xxx
-perm -xxx  # 相当于 x and x and x
-perm +xxx  # 相当于 x or x or 
```

关于时间
```
----- +4 ----->|4|------ -4 ----------- >
---------------------------------------->
-mtime 4
-mtime +4  # 4天之前
-mtime -4  # 4天之内
```
